My Skillset

I started self-teaching me Java while I was in school. That was also when I learned to use Programming IDEs like IntelliJ and Eclipse.
I designed graphics for my projects such as application icons, banners and UI layouts using software such as Inkscape, GIMP and Adobe Illustrator.
The background images on this website were built using the modeling software Blender.

Version Control Systems like Git are not only a part of my job but also almost all of my personal projects.
I work with a handful of Java Frameworks and libraries in my daily job such as JAX-RS, JPA, JAXB, Hibernate, JUnit, Mockito and Guice.
I used VueJS for both some of my private and work-based Web-based projects.
Other tools I regularly work with are Docker, Kubernetes, Portainer, Jenkins and the Apache Tomcat Server.

I also helped to design, build and manage a productive Kubernetes Cluster with automatic Gitlab Pipelines and deployment Helm Templates. This includes multiple HA applications and Microservices.
Technologies and Tools used include Rancher, Helm, Istio and other different native Kubernetes components.
