My Skillset

In my job as a Cloud Application Architect, Cloud System Engineer and Full-stack Developer I work with self-hosted and self-managed Kubernetes Clusters running multiple productive applications and microservices for hundreds of customers.
Tools that I regularly work with in this context are Docker, Kubernetes, Rancher, Istio, Helm and ArgoCD.

I am involved in the requirement analysis, design, implementation and deployment of these applications.
My deep knowledge of the Kubernetes Cluster architecture and application architecture patterns and best practices helps me to create scalable and maintainable applications.

When developing applications I work with a handful of Java Frameworks and libraries such as Spring Boot, JAX-RS, JPA, JAXB, Hibernate, JUnit, Mockito and Guice.
I am also using Go(lang) for performance critical applications.
I use VueJS for both some of my private and work-based Web-based projects.
For Continuous Integration and Continuous Delivery I use Jenkins and Gitlab CI/CD pipelines as well as ArgoCD.
